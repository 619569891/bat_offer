# interview_demo
2019年互联网大厂高频重点面试题Demo

资料来自于B站，【真全栈程序员】 公开的视频学习资料。
视频地址：https://www.bilibili.com/video/av68723403

由于demo演示需要用到java包，故建立一个简易版的springboot工程，依赖需要：lombok插件。

启动该demo必备：
1. jdk1.8_xx
1. maven 3.6.x
2. lombok插件
